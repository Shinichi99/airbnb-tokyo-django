Airbnb Price Optimization in Tokyo

Overview of the project

The prices of accommodation facilities can vary significantly depending on the season and the socio-economic situation. Therefore, appropriate pricing is crucial to strike a balance between profit margins and reservation rates. In this project, we developed an app to calculate the optimal prices for Airbnb accommodation in Tokyo. This app is based on the latest data from September 2023, and by providing information about the accommodation, users can obtain predictions of the suitable price using a machine learning model. This model, which utilizes data from approximately 10,000 accommodation facilities, is built on the XGBoost machine learning model, delivering a high level of prediction accuracy (the coefficient of determination:0.7).

Exploratory Data Analysis
Difficulty in the price prediction

The Airbnb data includes a wide range of prices, and individual hosts determine these prices. This variability can make predicting prices challenging. However, I utilize the relationships between price and other attributes as much as possible in my model.

Feature Engineering
Transport Accessibility

The location of the accommodation is one of the most important factors in determining the price. In this project, I use the distance from the nearest station as a feature. I also used the distances to the main railway stations in Tokyo.

Model Building
Machine Learning

The model is built on the XGBoost machine learning model. Its hyperparameters are tuned with Optuna. The model is trained with the data from approximately 10,000 accommodation facilities in Tokyo. The coefficient of determination is 0.7, which is a high level of prediction accuracy. However, the model is not perfect especially the high-priced accommodation facilities are not predicted well.

Web Deployment
A Variety of Tools

The model is deployed on Render. The app is built with Django, which is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. I also used Bootstrap for an attractive app design
